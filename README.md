An intelligent terminal assistant that generates code snippets and provides AI-powered explanations for your commands. Part of the **GitHub 30-Day Enrichment Plan**.

## 🚀 Features

- **Multi-language Code Generation**: Generate code in Python, Java, C, JavaScript, and Bash
- **AI-Powered Explanations**: Get clear explanations for generated code
- **Smart Suggestions**: Receive improvement suggestions for your code
- **Web Interface**: User-friendly Next.js frontend with responsive design
- **Extensible Architecture**: Easy to add new languages and features

## 🛠️ Installation

### Prerequisites
- Python 3.8+
- Node.js 18+
- npm or yarn

### Backend Setup
```bash
cd backend
# No additional dependencies needed for MVP
python main.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 💻 Usage

### Web Interface
1. Start the frontend: `npm run dev`
2. Open your browser to `http://localhost:3000`
3. Enter your command or request in the input field
4. Get AI-generated code and explanations

### Backend API (Future)
The backend is designed to be extended with a proper API endpoint that the frontend can consume.
